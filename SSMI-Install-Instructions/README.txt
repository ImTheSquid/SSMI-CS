== Quick Install Instructions ==
1) Move files to where you want them (you will not be able to move them after you install the service)
2) Run SSMIHelper.exe and login to Spotify
3) Run svcInstall.bat AS ADMINISTRATOR
4) Restart
You're done! If you want to move the service exe to somewhere else, you must reinstall (uninstall then install) the service.
 
 == Quick Uninstall Instructions ==
1) Run svcUninstall.bat AS ADMINISTRATOR
2) Restart
